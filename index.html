    • Lyvi

    A simple command-line lyrics viewer for various audio players

    Project maintained by ok100 Hosted on GitHub Pages — Theme by mattgraham

    Features:

    • Mouse support
    • Displays lyrics, artist info and guitar tabs
    • Optionally set terminal background to artist photo or album cover
    • More than 20 providers (thanks to glyr)
    • Player control with configurable keybindings or statusbar buttons
    • Cache for downloaded metadata
    • Configuration file

    Supported players:

    • Audacious
    • C* Music Player
    • DeaDBeeF
    • Gmusicbrowser
    • Guayadeque
    • MOC
    • mpg123
    • MPlayer
    • Music Player Daemon
    • pianobar
    • Qmmp
    • shell-fm
    • VLC Media Player
